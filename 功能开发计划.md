## 🛠️ 技术栈

### 前端
- **框架**: Vue 3 (Composition API)
- **构建工具**: Vite
- **UI框架**: Tailwind CSS
- **状态管理**: Pinia
- **路由**: Vue Router
- **HTTP客户端**: Axios

### 后端
- **框架**: Flask
- **ORM**: SQLAlchemy
- **数据库**: PostgreSQL (推荐) / SQLite
- **AI集成**: LangChain + OpenAI
- **向量数据库**: ChromaDB

---

## 📝 注意事项

1. **前后端分离**: 前端端口默认 5173，后端端口默认 5000
2. **API文档**: 查看 `movie_back/API接口文档.txt` 了解详细接口规范
3. **环境配置**: 参考 `movie_back/.env.example` 配置环境变量
4. **数据模型**: 查看 `movie_back/src/models/` 了解数据库结构
5. **路由配置**: 查看 `movie_front/src/router/index.js` 了解前端路由

## ✅ 快速开始

### 前端启动
```bash
cd movie_front
npm install
npm run dev
```

### 后端启动
```bash
cd movie_back
pip install -r requirements.txt
python app.py
```
